Certainly. The core idea behind Self-Attention in Transformers is to allow the model to weigh the importance of different words in a sentence relative to each other.

Think of it as a retrieval system. For every token (word), we generate three vectors:

  • Query (Q): What the token is looking for.
  • Key (K): What the token contains/offers.
  • Value (V): The actual information content of the token.

The attention score is calculated by taking the dot product of the Query and the Key.

import numpy as np
def scaled_dot_product_attention(Q, K, V):
    d_k = Q.shape[-1]
    scores = np.matmul(Q, K.T) / np.sqrt(d_k)
    weights = softmax(scores)
    return np.matmul(weights, V)

The result determines how much focus to put on other parts of the input sequence when encoding the current token.
